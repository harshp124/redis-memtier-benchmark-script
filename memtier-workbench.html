Parameter Help

Threads

Worker threads inside memtier. Not the same as clients.

Clients per thread

Connections opened by each thread. Total connections = threads × clients.

Pipeline

Number of requests in flight per connection. Lower values help isolate latency.

Ratio

Write:read mix. For 20% writes and 80% reads, memtier uses 1:4.

Value size

Controls payload size with --data-size or --data-size-range.

Key settings

Use key min, key max, key pattern, and key prefix to shape the keyspace.

Percentiles

Controls which latency percentiles appear in the summary table.

HDR prefix

Saves histogram files for deeper latency analysis.
